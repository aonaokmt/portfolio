カタログデザイン作成 および、ディレクション

目的
営業活動に活用するため、従来の時代に合わないカタログを刷新し、新規顧客に企業の魅力を的確に伝える。
取り組み
  1. ・チームで策定したデザインコンセプトを元に、デザインカンプ作成。
  2. ・Illustrator、Photoshopの実制作はフリーランスデザイナーへ委託し、ディレクションを担当。
  3. ・カラーコード等デザイン、写真選定などを複数回打合せ、最終的なカタログを完成。
  4. ＊使用ツール/Figma
成果
企業イメージを刷新し、営業活動において「会社の顔」となるツールを提供。展示会での配布や、SNSの寺院フォロワーに向けたDM送付といった新規顧客獲得に資する基盤を構築。

プロジェクト進行資料企画・作成

目的
新規事業の売上拡大に向け、企業ブランディングを基盤にした事業推進を図る。
取り組み
  1. ・5名のチームで事業を推進し、全員が共通認識を持てるよう現状整理を行った進行資料を作成。初回ミーティングで活用し、今後のアクションプランも提示。
  2. ・競合調査を通じて他社との差別化要因を明確化し、企業の強みを整理。
  3. ・ぺルソナ設定を行い、ターゲット顧客像を具体化することで「誰のための商品か」をチーム全体で共有。
  4. ＊使用ツール/PowerPoint
成果
事業初期段階から方向性を可視化し、チームが一丸となって取り組む体制を構築。今後の売上拡大に向けた基盤づくりに貢献。

営業ツール作成（営業資料、POP）

目的
新規顧客獲得のため、企業の魅力や商品のこだわりを効果的に伝える資料を制作
工夫
  1. ・主力商品の「クラフト酢」の首都圏販路拡大を意識し、首都圏向けの洗練されたデザインを採用しつつ、創業135年の歴史や伊勢神宮への奉納実績など老舗としての信頼性を打ち出した。
  2. ・企業カラーである朱色を配色に取り入れ、資料と企業イメージの統一感を演出。
  3. ・商品の強みや製造背景を視覚的に理解しやすいよう図表やレイアウトを工夫し訴求力を高めた。
  4. ＊使用ツール/PowerPoint
成果
商談での活用により、約半年間で新規4社6ヶ所の取引先開拓に成功。

  4. ・予算・実績表を作成・更新し、財務管理面でも事業を支援。
  5. ＊使用ツール/Googleスライド、スプレッドシート
